🍔 Restaurant-booking-system
A full-stack Django web application for restaurant management featuring online table booking, dynamic menu display, and upcoming automated email notifications.

🚀 Features
🪑 Online Table Booking System
🍕 Dynamic Menu Display (Burgers, Pizza, Momos, Fries)
📧 Email Notification (Booking Confirmation)
🧾 Customer Feedback Section
⚡ Fast and Responsive UI
🛠️ Tech Stack
Backend: Django (Python)
Frontend: HTML, CSS, JavaScript
Database: SQLite
Tools: Git, GitHub
📂 Project Structure
Restaurant_Project/ │── Restaurant_Project/ # Main project settings │── Base_App/ # Main app (views, models, templates) │── templates/ │── static/ │── manage.py

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/AyushMishra9793 Restaurant-booking-system.git
cd Restaurant-booking-system

2️⃣ Create virtual environment
python -m venv myenv
myenv\Scripts\activate   # Windows
3️⃣ Install dependencies
pip install -r requirements.txt
4️⃣ Run server
python manage.py runserver
🔐 Environment Variables

Create a .env file in root directory:

SECRET_KEY=your_secret_key
EMAIL_HOST_USER=your_email@gmail.com
EMAIL_HOST_PASSWORD=your_app_password
